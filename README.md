📊 Tech Layoffs Data Analysis (2020-2023)

📝 Overview
This project is an end-to-end data analysis focusing on global tech layoffs from 2020 to 2023. I performed data cleaning and exploratory analysis using **SQL (MySQL)**, then built an interactive dashboard in **Power BI** to visualize key trends and insights.

🛠️ Tools & Technologies
- **SQL (MySQL):** Data cleaning and Exploratory Data Analysis (EDA).
- **Power BI:** Data modeling and interactive dashboard creation.
- **Figma:** Designing the UI/UX layout for the dashboard.

🔍 Key Insights from the Analysis
- **Total Layoffs:** Over 121,000 employees were affected globally.
- **Top Sectors:** Retail and Consumer industries faced the highest impact.
- **Major Companies:** Analysis includes deep dives into layoffs at Amazon, Meta, and Google.
- **Timeline:** Visualized the layoffs trend month-by-month to identify peak periods.

📁 Repository Content
- `Data_Cleaning.sql`: SQL scripts for handling nulls, duplicates, and formatting.
- `Exploratory_Data_Analysis.sql`: Queries used for deep data exploration.
- `Layoffs_Analysis_Project.pdf`: A quick PDF preview of the final dashboard.
- `Layoffs_Analysis_Project.pbix`: The original Power BI file.
- `layoffs_full_data.csv`: The raw dataset used for the project.
